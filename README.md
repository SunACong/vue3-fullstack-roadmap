# 🚀 Vue3 Fullstack Roadmap

一个为期 6 个月的 Vue3 + TypeScript + 全栈开发进阶计划 📘  
目标：系统掌握 Vue3 生态，构建具备面试竞争力的项目作品集，实现前端到全栈的能力跃迁 💼

---

## 📌 项目背景

本仓库用于记录我从 Vue2 向 Vue3 + TypeScript 转型，并逐步拓展后端能力，最终实现全栈跳槽的全过程。

- ⏱ 每天学习约 3 小时
- 📅 总周期：半年（阶段一为 1~2 个月）
- 🎯 目标岗位：Vue 全栈 / 前端工程师

---
